# Mini Task Tracker (React)

This is a simple **Task Tracker application** built using **React** and **Tailwind CSS**.  
It allows users to manage tasks easily with a clean and responsive user interface.

This project uses **mock data only**.  
There is **no backend**, **no API**, and **no localStorage**.

---

## What This App Does

- Shows a list of tasks
- Each task has:
  - Title
  - Due Date
  - Status (Pending / Done)

- Add a new task
- Edit an existing task
- Delete a task
- Change task status using a **Done / Pending button**
- Filter tasks:
  - All
  - Pending
  - Done
- Search tasks by title (with debounce)
- Sort tasks:
  - By name
  - By due date
- Shows a message when **no task is found**

---

## Technologies Used

- React
- React Hooks
- Tailwind CSS
